      # 总结

      • 题目

      使用 React 开发一个 Button 组件, Button 组件具备向服务端发起请求, 并获取数据的能力, 要求在请求响应期间, 用户不能通过 Button 再次发起请求, 在此基础上尽可能的考虑, 需要有完整的 UI 测试, 单元测试, 测试框架使用 jest

      • 用户体验
      • 容错
      • 可复用
      • 扩展性
