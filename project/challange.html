          # 网吧上网登记数据可视化分析

          周志秀，赵银，周仕幸，蒋哲臣

          图1 系统界面图：(a)平行坐标系用于展示上网记录的完整信息，有利于发现感兴趣的模式并对相应的维度进行分析。(b)为外来人口特征分析图，用于分析外来人口的结构和维度。(c)地图用于展示网吧的大致分布情况以及相应的上网人数。可以选择感兴趣的网吧，从多个维度对其进行展示分析，以得到网吧的用户画像。 摘 要—基于本次比赛提供的市内3000多家网吧基本信息及2016年10-12三个月约1600万行上网记录，对上网人员的关系以及合法性进行分析。本文针对上网数据属性中的年龄、上网频率、上网时长、人员籍贯对数据进行初步筛选，结合气泡图、柱状图和平行坐标系来展示疑似未成年人上网信息和一些非法经营的网吧信息。通过对网民籍贯的分析，推断流动人口的行为特点。通过上网时空关系推断用户之间可能存在的联系，寻找可能存在的社团。 关键词—可视分析;数据筛选;流动人口；时空关系

          # 1.引言

          ChinaVis 2017提供了某市3000多个网吧基本信息及2016年三个月约1600万行上网记录的数据包，它记录了网吧编号、网吧名称、网吧位置信息等，上网人员信息包括上网编号、姓名、性别、籍贯、出生年年月、上下线时间。通过对上网人员的基本信息进行筛选分析，完成以下几个工作：1.挖掘出未成年人上网的信息及接纳未成年人上网的网吧信息；2.分析流动人口的上网记录总结他们的行为特点；3.从上网记录中发现社团现象；4.为网吧设计用户画像，帮助公安人员和用户甄别网吧的特性。

          # 2.概述

          首先对数据进行清洗，依据问题所需针对上网人员年龄、上网频率、上网时长、人员籍贯对数据进行筛选，适当调整后将其进行可视分析。在可视化设计阶段主要使用平行坐标系、气泡图和柱状图等发现未成年人上网信息和非法经营网吧信息。

          # 3 未成年人上网和非法网吧识别

          由于当前上网必须使用身份证登记，我们可以确定数据中使用到的身份证本身是合法的证件，因此，我们把目标锁定在一些年龄、上下线时间点、上网时常异常的数据上。我们筛选出部分上网人员数据，他们的年龄在50岁以上，上网次数较多，上网时间集中在中午和傍晚以及周末，即与学生的作息时间高度吻合。我们将这些数据记录列为未成年人冒用他人信息上网的行为记录。此外，我们发现某些身份证的连续上网时长超过了100小时，我们 怀疑这些身份证是网吧经营者自身所有的，经营者利用这些身份证件为未成年人进行上网信息的登记。

          # 4 可视化设计

          # 4.1 平行坐标

          对上网年龄超过50岁的网民进行分析，发现这些人的上网时间分布在工作日，且集中在午休或下午傍晚时间，双休日上网时间长，上线时间晚，这些特点与未成年人类似，可以将该人群列为未成年人上网人群，同时存在大量单次上网记录超过1000多个小时的网吧可以列为非法网吧。

          # 4.2 气泡图与柱状图相结合

          用图4.2所示的气泡图来展示网民不同上网频次代表的人数，柱状图展示了不同上网时长代表的人数，分析这两幅图可以筛选出上网异常的数据信息。

          # 4.3 树状图外籍人员结构和纬度

          我们首先按籍贯对外籍人员进行分类，使用树状图表示信息的结构层级，再对每一个籍贯按上网时长，上线时间点，男女比，年龄比进行相应的统计分析，以得到外籍上网人员的行为模式。

          # 4.4 框选与柱状图

          用图4.4表示所有网吧的地理位置，用匡选筛选网吧，并显示上网人次排在top20的网吧上网人次统计，接下来可选中相应的网吧看更多维度的信息，对这些信息进行分析以得出网吧用户画像。

          # 4.5 展示网吧维度的组件

          如图4.5所示以饼图、环形图以及柱状图展示选中网吧的本地籍贯与外地各个籍贯的占比、一天各个时刻上网人数占比以及上网年龄段和相应的性别分布。

          # 4.6 geiph展示社团关系

          我们用Geiph展示社团关系图，图中可以看出连接紧密的点群即为相关点，并且可以使用Gephi自带的属性过滤和统计得出进一步的信息。

          # 5.讨论

          本文从网吧上网记录出发，结合实际分析需求，设计出了较为合适的可视方案。

          为了寻找未成年人上网记录，本文结合上网时长，频率，年龄以及上网时间，得到某些高年龄，高频上网的人，且这些人的上网时间与未成年人的上网时间较为吻合，取这些身份证为未成年人上网身份证，并结合该身份证的上网记录具体信息进行分析可以知道该人为未成年人的可能性较大。

          在分析外来人口上网记录信息时，我们用树状图结构化的展示了外来人口的构成，用各种统计图展示了外来人口上网的行为特点，有效的满足了分析需求。 为了找出社团，我们以上网时间的吻合度作为最重要的向量，辅助以年龄，性别，籍贯等维度计算出上网人物关系的紧密程度。采用Geiph将结果展现出来，使结果能够更好的展示和分析。

          为了满足对各个网吧的维度进行展现和分析的需求。我们先在地图给出了网吧的分布，允许用户匡选相应的网吧并给出选中网吧的上网人次排序，以便用户从中挑选数据较多网吧。选中单个网吧后，系统能够详尽的展示该网吧其他维度的统计信息，方便用户对其进行分析以便得到用户画像。

          综上，此系统较好的满足了实际问题需求，有较好的实用性。
