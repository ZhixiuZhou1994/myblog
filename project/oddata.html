# 介绍

示例对出租车数据起始点数据的可视化展示分析

2016年刚进入实验室做的web小项目，在高德地图上分析出租出的起始点数据，在ProcessJson.java中数据的读取路径写死了，大家可以修改成自己下载下来data的文件的绝对路径

之前的项目，现在运行发现地图样式转化不过来了，哭唧唧

可视化分析组件：

  • Aggregate
  • OData
  • LensO
  • LensD
  • Clear
